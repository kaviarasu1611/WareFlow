📦 WareFlow

A simple and responsive Warehouse Management System website built using HTML, CSS, and JavaScript. This project demonstrates the basic functionalities and design of a warehouse management platform with a modern and user-friendly interface.

🚀 Features
🏠 Responsive Home Page
📦 Warehouse Collection Section
🚚 Transport Information
⭐ Reviews Section
📊 Warehouse Statistics
📱 Mobile-Friendly Navigation Menu
📞 Contact Page
🎨 Clean and Modern UI
⚡ Interactive JavaScript Features
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Google Fonts (Poppins)
Font Awesome

▶️ How to Run
Download or clone this repository.
Open the project folder.
Double-click index.html or open it in your web browser.
